# Agrimo About Page

This project is a static website layout inspired by the Agrimo agriculture theme.
It is built using HTML, CSS, and JavaScript and includes multiple sections such as About, Services, Testimonials, and Footer.

## Features

- Navigation bar with icons
- About section with image and statistics card
- Services section with card slider
- Customer testimonial section
- Footer with contact information and social icons

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
